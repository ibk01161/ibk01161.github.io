# 파이썬 코드 테스트2

```python
names = ["Joseph", "Peter", "Cook", "Tim"]  
  
print('Current names List is:', names)  
  
new_name = input("Please enter a name:\n")  
names.append(new_name)  # Using the append() function  
  
print('Updated name List is:', names)
```
